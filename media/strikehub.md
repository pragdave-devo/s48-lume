Demos of StrikeHub's core features — launching connectors, navigating between apps, and the authentication flow.

<div style="display: grid; grid-template-columns: repeat(auto-fill, minmax(280px, 1fr)); gap: 1.5rem; margin: 1.5rem 0;">
  **Launching Connectors**

Starting KubeStudio and Pick connectors from the sidebar. Watch the connector discovery process, health checks, and the transition from offline to online status.
  **Switching Apps**

Navigating between connector apps in the embedded content area. See how StrikeHub loads each connector's UI via the custom protocol handler and maintains state across switches.
  **Authentication Flow**

The complete OIDC sign-in flow: browser redirect to Keycloak, credential entry, callback handling, and automatic token injection into connector HTML.
