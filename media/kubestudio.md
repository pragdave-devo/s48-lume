Demos of KubeStudio's Kubernetes management features — cluster browsing, workload inspection, log streaming, scaling, and multi-cluster support.

<div style="display: grid; grid-template-columns: repeat(auto-fill, minmax(280px, 1fr)); gap: 1.5rem; margin: 1.5rem 0;">
  **Cluster Overview**

Browsing namespaces and viewing the cluster-wide resource summary. See node status, resource utilization, and namespace-level breakdowns at a glance.
  **Workload Browser**

Navigating deployments, pods, and services within a namespace. Drill into pod details, view container specs, and inspect resource requests and limits.
  **Pod Logs**

Streaming real-time logs from running containers. Follow log output live, filter by keyword, and switch between containers in multi-container pods.
  **Deployment Scaling**

Scaling replica counts and watching rollout progress. See pods spin up in real-time, monitor readiness probes, and track rollout status.
  **Multi-Cluster Switching**

Switching between kubeconfig contexts for different clusters. See how KubeStudio handles context changes, credential refresh, and workspace state.
</div>
