## Overview

The Assistant is an intelligent helper that:
- Understands your current engagement
- Provides context-aware recommendations
- Answers questions about features and workflows
- Assists with analysis and planning
- Helps interpret findings and data

## Accessing the Assistant

Navigate to **Headquarters → Assistant** to open the AI assistant interface.

![Assistant Interface](../images/Assistant.png)
*AI-powered assistant with conversational interface*

The Assistant is always available, regardless of whether you have an active engagement selected.

## Features

- **Context-aware**: Understands your engagement and current work
- **Conversational**: Natural language interaction
- **Actionable**: Provides specific recommendations
- **Educational**: Explains concepts and workflows

## Common Uses

- Get recommendations on next steps
- Ask questions about features
- Request analysis of your data
- Get help planning engagements
- Review findings for completeness
