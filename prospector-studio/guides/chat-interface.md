## Starting a Conversation

From the Studio UI, select an agent and open a new conversation. You can:

- Send messages and receive AI-generated responses
- Watch agents invoke tools in real-time
- View source citations from attached knowledge bases
- Trigger workflows directly from the conversation

## Conversations

Each conversation with an agent maintains full context history. Agents can handle multi-turn interactions, reference previous messages, and maintain state across a session.

## Tool Usage

During a conversation, agents can autonomously decide to invoke tools based on the user's request. Tool calls and their results are displayed inline, giving full visibility into the agent's reasoning and actions.
